# BMI Calculator in Python

This is a simple Python program that calculates Body Mass Index (BMI) 
based on weight in pounds and height in inches. It also categorizes BMI 
into standard health categories.

## Features
- Takes user input for name, weight, and height
- Calculates BMI using `(weight * 703) / (height^2)`
- Categorizes BMI:
  - Underweight
  - Normal weight
  - Overweight
  - Obese
  - Severely obese
  - Morbidly obese

## How to Run
1. Make sure Python 3 is installed
2. Run in terminal or IDE:
```bash
python BMI_Calculator.py
